# SlickMensWardrobe
Men's Apperal and Fragrance

# How to Install
```
npm install
```
# How to Start
```
node app.js
```


Typescript, Node, Postgresql


# Directories
```
/SLICKMENWARDROBE
│
├── /node_modules       # NPM 패키지들
│
├── /src                # TypeScript 소스 코드
│   ├── /configs
│   ├── /controllers
│   ├── /middlewares
│   ├── /modules
│   ├── /routes
│   ├── /utils
│   └── app.js
│
├── /dist               # 컴파일된 JavaScript 코드 (예: tsc 명령으로 생성)
│
├── package.json        # 프로젝트 메타데이터 및 의존성 정보
├── tsconfig.json       # TypeScript 설정
└── ...                 # 기타 설정 및 문서 파일

```
